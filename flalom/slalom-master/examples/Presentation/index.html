  • Gravitas
  • Simple physics simulations
  • Three basic simulations
    • Friction
    • Gravity/Constant Acceleration
    • Spring
  • Simple equations, no numerical integration
  • Assemble a simulation or reuse one, give it velocity and extents, then let it run!
  • All code and examples on github, licensed under APSLv2
  • Physics help touch interfaces create an intuitively real-feeling world
  • Physics improve coherence of interface by removing discontinuous motion
  • Physics communicate edges and boundaries unambiguously
  • All of those demos had hand-coded touch handlers
  • Many had hand-coded combinations of simulations
  • Boring, error-prone code slows iteration of design ideas
